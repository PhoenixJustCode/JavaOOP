# Problem 1: Questions and Answers

1. **What is the core difference between a class and an interface?**  
   A class defines state and behavior and can be instantiated. An interface defines a contract (behavior) and cannot be instantiated.

2. **Can interfaces have fields? If yes, what are the implicit modifiers for them?**  
   Yes. Fields in interfaces are implicitly `public static final`.

3. **Can a class implement multiple interfaces? Explain why this is useful.**  
   Yes. It allows a class to satisfy multiple contracts and exhibit different behaviors without the complexity of multiple class inheritance.
